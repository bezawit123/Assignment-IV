Pseudo-classes: A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s). For example, :hover can be used to change a button's color when the user's pointer hovers over it.

example

The following example will color all even rows gray (#ccc), all uneven rows which aren't dividable by 5 white and every other row magenta.

table tr:nth-child(2n) td{
background-color: #ccc;
}
table tr:nth-child(2n+1) td{
background-color: #fff;
}
table tr:nth-child(2n+1):nth-child(5n) td{
background-color: #f0f;
}

Pseudo-elements: A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element(s). For example, ::first-line can be used to change the font of the first line of a paragraph.

example

The following example will add a language-tag to every quote on a given page using the pseudo-class :lang and the pseudo-element ::after:

q:lang(de)::after{
content: " (German) ";
}
q:lang(en)::after{
content: " (English) ";
}
q:lang(fr)::after{
content: " (French) ";
}
q:not(:lang(fr)):not(:lang(de)):not(:lang(en))::after{
content: " (Unrecognized language) ";
}

#5-

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

  • An HTML element becomes a grid container when its display property is set to grid or inline-grid.

    .grid-container { display: grid; }

  • The grid-column-gap property sets the gap between the columns

    .grid-container { display: grid; grid-column-gap: 50px; }

  • The grid-gap property is a shorthand property for the grid-row-gap and the grid-column-gap properties

    .grid-container { display: grid; grid-gap: 50px 100px; }

  • The grid-gap property can also be used to set both the row gap and the column gap in one value

    .grid-container { display: grid; grid-gap: 50px; }

  • Place a grid item at column line 1, and let it end on column line 3

    .item1 { grid-column-start: 1; grid-column-end: 3; }

  • Place a grid item at row line 1, and let it end on row line 3

    .item1 { grid-row-start: 1; grid-row-end: 3; }

#6 -

CSS Grid and Flexbox are layout models that share similarities and can be used together. The key difference is that CSS Grid can be used to create two-dimensional layouts, while Flexbox can only be used to create one-dimensional layouts. That means you can place components along the X- and Y-axis in CSS Grid and only one axis in Flexbox.
